Create a chord diagram by inserting a <uke-chord> element in your HTML. Set the frets attribute to the desired frets that should be played. An F chord for example would have frets set to 2010

To add fingerings use a fingers attribute

To add a chord name use a `name` attribute. To add text under each string, such as root position indicators use a `sub` attribute with 0's for blanks. This text is currently limited to one character per string. To show the nut use the `nut` attribute.

Make the diagrams larger by passing a `size="L"` attribute. Show the root on the diagram with a diamond head by passing a `root` attribute set to the string where the root of the chord is found. For example a root on the 3rd string would look like this: `root=3` attribute.
